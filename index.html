📖 Tutorial Lengkap Membuat Gamepass:
1
Buka create.roblox.com di browser kamu
2
Login dengan akun Roblox kamu
3
Klik salah satu game kamu (kalau belum punya, buat dulu game kosong)
4
Di sidebar kiri klik "Monetization" → lalu klik "Passes"
5
Klik tombol "Create a Pass"
6
Upload foto bebas → isi nama Gamepass bebas → klik "Create"
7
Klik "..." di Gamepass → "Edit" → aktifkan "Item for Sale" → set harga 72 Robux → klik "Save"
